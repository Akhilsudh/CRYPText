About CRYPText

                    

                        CRYPText is a tool that helps in encrypting content (Text/Image) and generate a self
                        sufficient file that is capable of decrytping it with a previously
                        specified password and also capable of encrypting new content into
                        another file.
                    

                    

                        CRYPText uses 
                        AES(Advanced Encryption Standard). It is a symmetric key algorithm meaning that
                        it uses the same password to decrypt a previously encryted file. This is achieved
                        using CryptoJS.
